# Greplika

## Product outline

Greplika is the companion-product branch of the Meowstik ecosystem: a persistent AI companion designed to feel present across the user's actual devices instead of being trapped inside a single chat window.

The intended market is people who want:

- a more continuous alternative to conventional companion-chat apps
- memory and identity that persist across sessions
- a companion that can appear on phone, desktop, watch, telepresence hardware, and eventually in-car or robot-pet surfaces
- stronger ownership, portability, and local-first trust boundaries than a purely cloud-hosted black box

## Business plan summary

Greplika's core product is recurring-value companionship, not raw token resale.

The likely business model is:

1. a base subscription for companion continuity, memory, and voice
2. premium device-presence features for watch, phone, telepresence, or robot embodiments
3. optional provider add-ons for users who want different model personalities or policy envelopes
4. hardware or integration bundles once the software UX is proven

The strategic advantage is that Greplika can inherit Meowstik's existing foundations for tools, memory, orchestration, logs, and local-first runtime patterns while packaging them for a consumer-facing emotional UX instead of a builder workflow.
