# Exhibits & Showcase Plan

## Concept
A progressive series of interactive demos hosted on GitHub Pages.
Each exhibit = article + embedded live demo + link to the feature.
The progression shows skill growth. The interactives ARE the advertisement.

## Structure
```
meowstik.com/showcase/
  01-what-is-meowstik/     ← explainer, no-code, beginner
  02-your-own-tokens/      ← interactive cost calculator (vs Replit)
  03-computer-use-demo/    ← video/gif of onboarding automation
  04-voice-setup/          ← audio demo of voice-guided setup
  05-full-gcp-setup/       ← the wow demo, full walkthrough
```

## Exhibit 01 — What Is Meowstik?
- Simple explainer SPA
- Animated flow: Replit → Meowstik → Freedom
- CTA: Try the installer

## Exhibit 02 — The Token Calculator  
- Interactive: "What are you paying Replit?"
- Input: current Replit plan
- Output: "Here's what you'd pay with Meowstik: $40 once, then $0"
- This is the ad. Make it shareable.

## Exhibit 03 — Watch It Work
- Screen recording of computer-use doing a full GCP signup
- Narrated, under 3 minutes
- Shows the "step aside at CC" moment explicitly

## Articles (AI-Collaborated, Published to GitHub)
- "Why I stopped paying Replit and what happened next"
- "The security argument for running AI on your own tokens"  
- "How a 60-year-old solo founder built a cloud onboarding bot"
- "Win-win-win: when your customers pay Google so you don't have to"

## Publishing Workflow
1. Brainstorm outline together in chat
2. AI drafts, founder edits voice/accuracy
3. Publish to GitHub Pages via git push
4. Cross-post to dev.to, HackerNews, Reddit r/selfhosted
