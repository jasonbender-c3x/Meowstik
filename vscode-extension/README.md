# Meowstik VS Code Extension

This extension integrates Meowstik AI into Visual Studio Code.

## Features

- **Meowstik: Start Assistant**: Opens the Meowstik AI interface in a sidebar or editor panel.

## Requirements

- Meowstik server must be running locally or be accessible via URL.
- Default configured URL is `http://localhost:5000`.

## Extension Settings

This extension contributes the following settings:

* `meowstik.serverUrl`: The URL of the Meowstik backend server.
