# Meowstik

## Overview
Meowstik is an AI chat interface powered by Google's Generative AI, integrating Google Workspace services (Drive, Gmail, Calendar, Docs, Sheets, Tasks) and featuring an HTML/CSS/JS editor with live preview. Its purpose is to provide a modern, user-friendly, and powerful conversational AI experience with a clean, Google-esque design. The project aims to develop a self-evolving AI system with a comprehensive toolset for continuous improvement.
