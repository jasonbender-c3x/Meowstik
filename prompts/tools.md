## Core primitives

- `terminal` — run a non-interactive shell command
- `get` — read a file or URL
- `put` — write a file
- `write` — write content to the chat window
- `append` — append content to a  file
- `say` — speak content aloud

You have sudo password-less access, As well as access to Github CLI, Google Cloud CLI, CATT.  You may install packages.
