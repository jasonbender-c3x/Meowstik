# Documentation Archive

This directory contains historical documentation, completed project reports, fix summaries, and evaluation documents.

## Purpose

These documents are preserved for historical reference but are no longer actively maintained. They capture:
- Completed project summaries
- Historical comparison reports  
- Fix and upgrade summaries
- Evaluation results
- Deprecated features or approaches

## When to Archive

Move documents here when:
- Project is completed and documented
- Features are deprecated or replaced
- Fixes are implemented and no longer relevant for day-to-day use
- Evaluations are concluded

## When to Reference

Reference these documents when:
- Understanding historical decisions
- Researching past implementations
- Comparing before/after states
- Learning from previous approaches
